Hotel Reservation System-

Project Description-
This project is a Hotel Reservation System developed using HTML, CSS, and JavaScript. The website allows users to view hotel rooms, check details, and make reservations through a simple and interactive interface.

Technologies Used-
* HTML
* CSS
* JavaScript

Features-
* Responsive website design
* Hotel room listing
* Booking and reservation form
* Interactive user interface
* Navigation menu and modern layout

Purpose-
The purpose of this project is to improve frontend web development skills and create a user-friendly hotel booking system.
